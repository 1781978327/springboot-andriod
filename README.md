# SpringBoot-Android 聊天应用

这是一个基于 SpringBoot 后端和 Android 客户端的智能聊天应用。

## 项目结构

- `ai_1/`: Android 客户端项目
  - 使用 Retrofit 进行网络请求
  - 支持 Markdown 渲染
  - 现代化的聊天界面设计
  - 代码高亮显示

- `springboot/`: SpringBoot 后端项目
  - RESTful API 设计
  - 智能对话处理

## 功能特点

- 实时对话
- Markdown 支持
- 代码高亮
- 优雅的 UI 设计
- 响应式布局

## 技术栈

### Android 端
- Retrofit
- Markwon
- Prism4j
- Material Design

### 后端
- SpringBoot
- RESTful API
